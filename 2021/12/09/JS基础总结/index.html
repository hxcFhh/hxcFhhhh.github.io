1.5.2 转换为Number类型

Number类型是值类型

将其他的类型转换为Number

转换方式一：

    使用Number()函数

    字符串→

        1.如果字符串中有非数字内容，则转换为NaN

        2.如果是空字符串，或者全是空格，则转换为0

        3.如果是纯数字，则转换成功

    Bool→

        1.true为1

        2.false为0

        Null–> 0

    undefinde → 数字NaN

转换方式二：

    parseInt()可以将一个字符换里有效的整数内容读取出来

ps：这里的转换方式二有一个不好的点，就是转换的进制不确定，但是这个函数是一个重载的函数，当你再加入一个参数用来表示进制的时候，这个问题就可以得到解决

在js中，如果需要表示16进制的数字，则需要以0x开头,如果是0开头的话就是8进制,如果要表示二进制数字，则需要以0b开头，但是不是所有的浏览器都支持,如果使用parseInt()，有些浏览器会解析成十进制，有些会解析成八进制,这个方法可以使用parseInt(a,10)，传入第二个参数解决这个问题，和go中的strconv.parseInt()一致

    首先要清楚一点，Boolean 在js中有没有用1或者0等数字来转换的规则

    注意不要将基本类型中的布尔值 true 和 false 与值为 true 和 false 的 Boolean 对象弄混了。
    
    其值不是undefined或null的任何对象（包括其值为false的布尔对象）在传递给条件语句时都将计算为true。

  1. 0，-0，null，false，NaN，undefined，或空字符串（""）如果转换为Boolean初始值的话是false。

  2. 其余的值，包括任何对象或者是空数组或者是字符串的”false”都是true

2.运算符

2.1 算术运算符

算数运算符

任何数据和NaN进行运算最后的结果都是NaN

对非Number值进行运算的时候都会转成Number值进行运算，string除外

+

    -可以对两个值进行加法运算，字符串相加，拼串操作

    -非Number类型的值进行加减法的时候，true = 1 ,false = 0, null = 0

    -任何值和字符串做加法都会先转换成字符串，然后进行拼串操作

    -任意类型加上一个空串即可变成字符串

-

    -两个值相减，减一个string也会转变成Number

*

    -同上，做加减乘除都会转换为Number，null为0，undefined为NaN

/

    -同上

%

    -同别的语言

2.2 一元运算符

一元运算符，只需要一个操作数

正号

    正号不会对数字产生任何影响

负号

    负号可以对数字进行符号的取反

-对于一个非Number类型的值

    它会将其先转换为Number类型的值，然后再运算

可以通过这个方式对于一个变量进行类型的转换，通过+号

2.3 自增自减运算符

自增 ++ - 通过自增可以改变+1

    - 和C语言一模一样

自减 — -通过自减可以改变-1

    - 和C语言一模一样

非 :

4.1.5 形参与实参

  1. 形参并不需要给定类型，所以在执行特定功能的函数时最好是去判断类型再决定是否继续执行

  2. 解析器并不会去管你有多少实参，如果是大于需要的参数，那么多余的实参会被忽略掉

  3. 如果是少于的话，没有实参的形参就是undefined

4.1.6 返回值

  1. js不能达到多个返回值的效果，如果想要做到多个返回值的效果可以使用全局变量或者是引用类型或者是数组的方式达到这个效果

  2. 立即执行函数需要使用括号包起来
