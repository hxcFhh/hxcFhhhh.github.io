1 vb.net简介与安装

VB.NET是什么：VB.Net是一种简单，现代，面向对象的计算机编程语言，由微软开发，将**.NET Framework和公共语言运行库的强大功能与作为Visual Basic**标志的生产力优势相结合

VB.NET跟其他的.NET语言一样，是完全面向对象编程的， VB.NET中的一切都是一个对象，包括所有的基本类型（Short，Integer，Long，String，Boolean等）和用户定义的类型，事件，甚至是程序集。 所有对象从基类Object继承。

1.1 VB.NET的优点

  • 布尔条件
  • 自动垃圾回收
  • 标准库
  • 程序集版本控制
  • 属性和事件
  • 委托和事件管理
  • 易于使用泛型
  • 索引器
  • 条件编译
  • 简单的多线程

1.2 VB和VB.NET之间的关系

VB.NET并不是VB的升级版，VB.NET是一门面向对象的语言，而VB仅仅是一门面向过程的语言，而且VB.NET基于.NET framework库，在语法结构上必然与VB有着很多的类似，但是这不能说VB.Net是VB的升级版

2 基础教程

2.1 程序结构

一个VB.Net程序主要由以下几部分组成：

  • 命名空间声明 Namespace declaration
  • 一个类或模块 A class or module
  • 一个或多个程序 One or more procedures
  • 变量 Variables
  • 主过程 The Main procedure
  • 语句和表达式 Statements & Expressions
  • 注释 Comments

程序解析：

  1. Imports System用于在程序中包括系统命名空间（栈空间）
  2. 下一行的module声明，声明了一个模块，因为是面向对象的语言，所以每个程序都必须包含一个类的模块
  3. 类或模块通常将包含多个过程。 过程包含可执行代码，或者换句话说，它们定义了类的行为。 过程可以是以下任何一种：
    1. 功能，又叫做函数 Function
    2. 子 sub，简单理解为模块中的子元素
    3. 运算符 Operator
    4. 获取 Get
    5. 组 Set
    6. AddHandler
    7. RemoveHandler
    8. raiseEvent
  4. 然后是Console.WriteLine()，Console是System包下面的一个方法类似于js中的Console方法
  5. 最后就是Console.ReadLine(),此方法类似于C最后的getchar，让程序等待读取到键盘输入时再退出。

2.2 基本语法

面向对象的一些基本的概念：

  1. Object对象： 对象有着自己的状态和行为。例如：定义一个狗类，有颜色，种类，名字，年龄，以及它的行为，摇尾巴或者狗叫，对象是类的实例化，而行为则是一个对象能做的事情，并且是只有他能做的事情，别人是做不了的，除非你给别人也定义一个相同的行为。
  2. Class类： 类可以被定义成任何可以被抽象的事物，例如猫、狗、人、数、课程、学校以及你所能抽象的事物，他们有自己的动作，也有自己的状态。
  3. Methods方法： 方法就是一种行为，是一个类特定的行为
  4. 实例变量： 每个对象都有唯一的实例变量集合，对象的状态分配给这些实例变量的值创建。

2.5.1 变量声明

Dim语句用于一个或者多个变量的声明和存储分配。

Dim语句用于模块、类、结构、过程或块级别

VB.Net 变量声明的语法是：

[] [accessmodifier] [shared] [shadows] [static] ]

[ReadOnly] Dim [WithEvents] variablelist

  1. attributelist是适用于变量的属性列表，可选
  2. accessmodifier定义变量的访问级别，它具有值Public，Protected，Friend，Protected Friend和Private。
  3. shared共享一个声明变量，它不与类或结构的任何特定实例相关联，而是可用于类或结构的所有的实例。

  6. ReadOnly表示变量可以读取，但不能写入，可选的。

2.14 日期与时间

你写的大部分软件都需要实现某种形式的日期功能，返回当前日期和时间。日期是日常生活的一部分，使用它能让工作变得轻松，不需要太多思考。 VB.Net还提供了强大的日期算术工具，使操作日期变得容易。

日期数据类型包含日期值，时间值或日期和时间值。 Date的默认值为0001年1月1日的0:00:00（午夜）。等效的.NET数据类型为System.DateTime。

DateTime 结构表示即时时间，通常表示为日期和时间的一天

您还可以从DateAndTime类获取当前日期和时间。

DateAndTime模块包含日期和时间操作中使用的过程和属性。

注意：DateTime结构和DateAndTime模块都包含诸如Now和Today之类的属性，因此初学者经常会感到困惑。 DateAndTime类属于Microsoft.VisualBasic命名空间，DateTime结构属于System命名空间。 因此，使用以后的将帮助你移植你的代码到另一个.Net语言，如C＃。 但是，DateAndTime类/模块包含Visual Basic中所有可用的旧日期函数。

2.14.3 格式化日期

日期字面值应该用哈希符号（＃＃）括起来，并以M / d / yyyy格式指定，例如＃12/16/2012＃。 否则，您的代码可能会更改，具体取决于运行应用程序的语言环境。

例如，您为2012年2月6日的日期指定了＃2/6/2012＃的日期字面值。对于使用mm / dd / yyyy格式的语言环境，这是正确的。 但是，在使用dd / mm / yyyy格式的语言环境中，您的文本将编译为2012年6月2日。如果语言环境使用另一种格式，例如yyyy / mm / dd，该文字将无效并导致编译器错误。

要将Date字面值转换为语言环境的格式或自定义格式，请使用String类的Format函数，指定预定义或用户定义的日期格式。
